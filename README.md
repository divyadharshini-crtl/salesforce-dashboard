# Salesforce Real-Time Dashboard

## Setup
1. Clone: `git clone https://github.com/divyadharshini-crtl/salesforce-dashboard.git`
2. `cd salesforce-dashboard`
3. `cp .env.example .env` → Fill your Salesforce credentials
4. `npm install`
5. `node server.js` (Backend - port 3001)
6. `npm run dev` (Frontend - port 9000)
7. Visit `http://localhost:3001/auth` to connect Salesforce
